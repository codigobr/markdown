Markdown Here

O Markdown Here é uma extensão para o Browser que permite converter um conteúdo Markdown digitado dentro de um campo de formulário do site para a formatação Rich Text.

  1. Markdown
  3. Editores Markdown
  5. Markdown Here

Visão geral

A extensão Markdown Here fornece ao usuário a opção de levar o Markdown aonde ele for.

No mínimo podemos dizer que a ferramenta é bem ambiciosa, já que, a quantidade de sites diferentes que existem por aí é astronômica, e o aplicativo se aventura em todos.

Em todos? Digo, os sites que fornecem suporte à formatação Rich Text, isso, aquele tipo de formatação do Microsoft Word como o negrito, itálico e o sublinhado.

Sites como o Gmail, Hotmail e WordPress são exemplos de locais onde o Markdown Here pode ser utilizado.

Mas não para por aí, o Markdown Here funciona até na versão Web do Evernote.

Clicando aqui você pode consultar uma lista de serviços compatíveis com a ferramenta.

Para usar o Markdown Here, primeiro instale a extensão para o seu navegador, visite um site como o Gmail, digite o conteúdo do seu email usando a formatação Markdown, clique com o botão contrário do mouse (botão direito) dentro do formulário da mensagem e clique em Markdown Toggle.

Veja na imagem abaixo.

Prontinho, o Markdown Here fará o trabalho de converter o seu conteúdo Markdown em Rich Text dentro do próprio site.

Claro que, pela quantidade absurda de sites que fornecem formatações diferente pode haver inconsistência no conteúdo renderizado.

Sempre revise a formatação da sua mensagem, verifique se está de acordo com o que você deseja antes de enviá-la a alguém.

O site oficial do Markdown Here fica localizado em https://markdown-here.com/.
