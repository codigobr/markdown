Standard Notes

O Standard Notes é um aplicativo de anotações simples, seguro e de código aberto.

  1. Markdown
  3. Editores Markdown
  5. Standard Notes

Visão geral

O Standard Notes é um aplicativo gratuito de anotações que oferece recursos excelentes ao usuário.

A ferramenta não possui suporte ao Markdown por padrão, mas você pode habilitá-lo ao comprar a licença estendida do aplicativo.

Com essa licença você tem acesso a diversas extensões para modificar o comportamento do programa, uma delas é o Advanced Markdown .

Os elementos Markdown listados nesse artigo são baseados nessa extensão.

Mas por que usar o Standard Notes?

Com uma variedade enorme de opção para você escolher entre aplicativos Markdown, por que usar o Standard Notes?

Bom, ele oferece ao usuário uma excelente experiência com recursos de sincronização, privacidade e diversas extensões.

A equipe do Standard Notes também é comprometida com a filosofia do código fonte aberto, dando mais transparência para o programa e aumentando a confiabilidade do projeto.

A compatibilidade é outro ponto forte do app, a maioria das plataformas possuem versões disponíveis para elas.

Você pode encontrar mais informações (em inglês) sobre o Standard Notes no site oficial https://standardnotes.org/ .
