Simplenote

Simplenote é um programa de anotações simples compatível com o Markdown.

  1. Markdown
  3. Editores Markdown
  5. Simplenote

Visão geral

O aplicativo de anotações Simplenote foi desenvolvido pela empresa Automattic, criadora do WordPress.

O Simplenote conta com versões para a web, rodando diretamente no seu navegador, além de versões para dispositivos moveis e desktop.

A compatibilidade é excelente também, o programa está disponívei para a maioria dos sistemas operacionais, dentre eles, Windows, macOS, linux e Android.

O Simplenote é grátis e código fonte aberto , você pode visitar o repositório da ferramenta no GitHub em https://github.com/Automattic/Simplenote-United e ter acesso ao código de todas as versões.

Para usar o programa é necessário criar uma conta, com isso, seus documentos serão sincronizados em todos os seus dispositivos através do servidor da Automattic.

É importante saber que a Automattic não oferece suporte a criptografia  dos seus arquivos e a opção de sincronização não pode ser desativada.

O Simplenote possui uma opção interessante de exportação chamada Publish to Web (Publicar para a Web), nela o usuário pode gerar uma URL pública para um documento e compartilhá-la com outras pessoas.

Habilitando o suporte ao Markdown no Simplenote

Você pode habilitar o suporte ao Markdown no Simplenote clicando no ícone de informações e selecionando a opção Markdown Formatted.

Quando você habilita essa opção ela se torna padrão a partir de agora, ou seja, as anotações poderão ser feitas usando o Markdown.

Para mais informações sobre o Simplenote você pode visitar o site oficial da ferramenta em https://simplenote.com/ .

Como pré-visualizar documentos Markdown no Simplenote

O Simplenote não possui um editor em tempo real, ou seja, as formatações do Markdown não são apresentadas enquanto o usuário digita.

Para visualizar a renderização de um documento Markdown no Simplenote, clique no ícone do olho no canto superior direito.

Suporte ao Markdown no Simplenote

A Tabela 1.1 mostra os elementos Markdown que são compatíveis com o Simplenote.

Elemento Suporte Obs
Títulos Sim
Parágrafos Sim
Quebras de linha Sim
Negrito Sim
Itálico Parcial O Simplenote não fornece suporte para o uso do sublinhado _ para formatar um texto em itálico, usando esse símbolo ao redor do conteúdo, o texto ficará com uma linha em baixo.
Bloco de citação Sim
Listas ordenadas Sim
Listas não ordenadas Sim
Código Parcial O Simplenote não é compatível com o bloco de código básico do Markdown, ao invés disso, use a sintaxe do bloco de código cercado.
Linhas horizontais Sim
Links Sim
Imagens Sim
Tabelas Sim
Blocos de código cercado Sim Realçar a sintaxe do código dentro do bloco não é possível no Simplenote.
Notas de rodapé Sim
Id de títulos Não
Listas de definições Não
Riscado Sim
Listas de tarefas Sim
Emoji (copiar e colar) Sim
Emoji (códigos de acesso) Não
Link automático de URL Sim
Desativar o link automático de URL Sim
HTML Não

Elementos Markdown adicionais no Simplenote

A Tabela 1.2 apresenta alguns elementos mais obscuros do Markdown que podem ser usados no Simplenote.

Elemento Markdown Saída renderizada
Realçar ::palavra ou frase:: palavra ou frase
Sobrescrito X^2^ X2
Sublinhado _palavra ou frase_ palavra ou frase
