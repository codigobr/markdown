Markdeep

O Markdeep é uma ferramenta gratuita e simples que transforma qualquer arquivo Markdown em um arquivo HTML independente que pode ser visualizado em um navegador da web.

  1. Markdown
  3. Ferramentas
  5. Markdeep

Visão geral

Não há nada para instalar e não há serviço para o qual se registrar - basta adicionar uma linha de código na parte inferior do seu arquivo Markdown. Essa é uma ótima opção se você precisar visualizar rapidamente um arquivo Markdown em um navegador da Web ou compartilhar um arquivo md com alguém que precise visualizar a saída renderizada.

O uso do Markdeep é um processo de três etapas:

  • Adicione o seguinte código em uma única linha na parte inferior de um arquivo Markdown. Isso ativará o Markdeep. 
    <!-- Markdeep: --><style class="fallback">body{visibility:hidden;white-space:pre;font-family:monospace}</style><script src="markdeep.min.js" charset="utf-8"></script><script src="https://casual-effects.com/markdeep/latest/markdeep.min.js" charset="utf-8"></script><script>window.alreadyProcessedMarkdeep||(document.body.style.visibility="visible")</script>
  • Renomeie o arquivo Markdown adicionando a extensão .md.html (por exemplo, meuArquivo.md.html).
  • Abra o arquivo em um navegador da web para ver a saída renderizada.

Essa ferramenta possui muitos recursos além dos que são apresentados nessa página. Por exemplo, você pode escolher entre uma variedade de modelos para personalizar a aparência da sua página. O Markdeep também suporta diagramas, tipografia LaTeX para equações e muito mais. Confira a excelente documentação para obter todos os detalhes.

Site: https://casual-effects.com/markdeep

Suporte do Markdown no Markdeep

O Markdeep fornece suporte para os seguintes elementos Markdown.

Elemento Suporte Obs
Títulos Sim
Parágrafos Sim
Quebras de linha Não
Negrito Parcial Asteriscos no meio de uma palavra (ou seja no**meio**da palavra) são renderizados literalmente.
Itálico Parcial Asteriscos no meio de uma palavra (ou seja no*meio*da palavra) são renderizados literalmente
Bloco de citação Sim
Listas ordenadas Sim
Listas não ordenadas Sim
Código Sim
Linhas horizontais Sim
Links Sim
Imagens Sim
Tabelas Sim
Blocos de código cercado Sim O realce da sintaxe também é suportado.
Notas de rodapé Sim
Id de títulos Não
Listas de definições Sim
Riscado Sim
Listas de tarefas Sim
Emoji (copiar e colar) Desconhecido

Desativar o link automático de URL Sim
