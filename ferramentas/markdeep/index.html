Markdeep

O Markdeep é um programa gratuito que converte um arquivo Markdown em HTML.

  1. Markdown
  3. Editores Markdown

Visão geral

Com o Markdeep você consegue converter rapidamente os seus documentos Markdown em HTML, e com isso, visualizar o arquivo em qualquer navegador Web.

Para usar o Markdeep não é preciso instalar absolutamente nada e muito menos se registrar em qualquer site.

O procedimento é feito adicionando apenas uma linha de código no final do seu arquivo Markdown.

Se você precisa visualizar rapidamente um arquivo em um Browser ou até mesmo compartilhar o documento para que outro usuário analise a saída, o Markdeep é a ferramenta ideal para essa tarefa.

Vamos entender como converter o arquivo Markdown em HTML usando o Markdeep.

  • Primeiro, adicione o código, mostrado abaixo, no final do seu documento Markdown. 
    <!-- Markdeep: --><style class="fallback">body{visibility:hidden;white-space:pre;font-family:monospace}</style><script src="markdeep.min.js" charset="utf-8"></script><script src="https://casual-effects.com/markdeep/latest/markdeep.min.js" charset="utf-8"></script><script>window.alreadyProcessedMarkdeep||(document.body.style.visibility="visible")</script>
  • Segundo, altere o nome do seu arquivo e adicione uma extensão html. Por exemplo, se o nome do seu arquivo é file.md renomeie ele para file.md.html.
  • Terceiro, carregue o arquivo em um navegador da Web como o Google Chrome ou o Firefox.

O Markdeep é bem generoso em opções do que você pode fazer com o seu documento. Existem vários modelos para você personalizar a aparência da sua página convertida em HTML; suporte para gerar diagramas; apresentação de tipografia LaTeX para equações matemáticas e muito mais.

Você pode encontrar mais informações sobre o Markdeep no site oficial da ferramenta https://casual-effects.com/markdeep.
