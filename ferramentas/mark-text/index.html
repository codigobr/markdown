Mark Text

O Mark Text é um editor de documentos gratuito e de código aberto projetado exclusivamente para escrita em Markdown.

  1. Markdown
  3. Ferramentas
  5. Mark Text

Visão geral

Como o Typora, o Mark Text possui uma interface polida e um editor em tempo real que oculta a formatação do Markdown após a digitação. As opções de exportação de PDF e HTML são úteis, assim como o recurso que permite copiar texto do editor no formato Markdown, HTML ou texto sem formatação.

Existem alguns probleminhas. Em várias instâncias (observadas abaixo na tabela), a aparência do texto no aplicativo não corresponde à saída renderizada do HTML e PDF exportado. E, como no Notion, pode ser difícil editar o texto no formato Markdown depois que o editor já o converteu.

Site: https://marktext.app/

Suporte do Markdown no Mark Text
