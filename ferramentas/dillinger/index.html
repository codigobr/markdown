Dillinger

O Dilliger é um editor Markdown prático, ele carrega diretamente no seu navegador, ou seja, não há a necessidade de instalar nenhum aplicativo no seu computador.

Introdução

O Dillinger possui um conversor em tempo real que mostra o resultado da formatação Markdown enquanto você digita.

A ferramenta é dividida em dois painéis, à esquerda você insere a formatação Markdown e à direita aparece a conversão em HTML.

Com essa visualização do seu documento formatado, você consegue, em tempo real, visualizar a aparência do texto final.

O suporte ao Markdown no Dillinger é excelente. A aplicação utiliza o processador Markdown makdown-it  para entender os marcadores.

No quesito exportação de documentos, bom, a ferramenta cumpre o seu papel, mas opções para exportar não são personalizáveis, o que pode ser um problema para usuários mais exigentes.

Outro fator importante a ser notado é que o Dillinger, por ser uma ferramenta online, necessita de uma conexão com a internet. Se algo der errado como uma queda de conectividade ou até mesmo o travamento do navegador, todos o seu conteúdo formatado pode ser perdido.

Sabendo disso, eu recomendo que você use o Dillinger de forma mais casual, para aprender o Markdown, testar marcações, fazer prévias de formatações, anotações rápidas, etc.

O site oficial do Dillinger fica localizado em https://dillinger.io/ .

Suporte ao Markdown no Dillinger

Na Tabela 1.1 você confere os elementos Markdown que são suportados pelo Dillinger.

Elemento Suporte Obs
Títulos Sim A ferramenta não fornece suporte à sintaxe alternativa de criação de títulos/cabeçalhos.
Parágrafos Sim
Quebras de linha Parcial Não fornece compatibilidade com o uso do espaço em branco a direta da linha para representar a quebra, para inserir uma nova linha pressione o Enter.
Negrito Sim
Itálico Sim
Bloco de citação Sim Blocos de citação aninhados não são suportados pelo Dillinger.
Listas ordenadas Sim
Listas não ordenadas Sim
Código Sim
Linhas horizontais Sim
Links Sim
Imagens Sim Não há a possibilidade de enviar imagens para o aplicativo, você terá que armazená-las em outro servidor e usar o endereço completo da imagem no seu documento.
Tabelas Sim
Blocos de código cercado Sim O realce da sintaxe também é suportado.
Notas de rodapé Sim
Id de títulos Parcial A identificação de título é gerada automaticamente pela ferramenta, ou seja, não há suporte para a personalização pelo usuário.
Listas de definições Sim
Riscado Sim
Listas de tarefas Sim Não precisa colocar o traço - no início da linha do elemento da lista, para criar um novo elemento somente o uso dos colchetes [] já é suficiente.
Emoji (copiar e colar) Sim
Emoji (códigos de acesso) Não
Link automático de URL Sim
Desativar o link automático de URL Não
HTML Não
