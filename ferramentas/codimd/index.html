CodiMD

O CodiMD é um editor de código fonte aberto que oferece o recurso de colaboração em tempo real, além é claro, de ser compatível com o Markdown.

Introdução

Uma ferramenta poderosa que permite que você colabore com a sua equipe de desenvolvimento em tempo real, e o melhor, usando o bom e velho Markdown para editar os documentos.

O CodiMD foi construído e baseado no código-fonte do HackMD.

Usando essa ferramenta você pode hospedar e controlar com facilidade o conteúdo do seu projeto.

O suporte ao Markdown no CodiMD é excelente, é compatível, inclusive, com a sintaxe do CommonMark.

Alguns plugins interessantes da ferramenta:

  • MathJax - Usando para renderizar fórmulas matemáticas.
  • Mermaid e Graphviz – Diagramação UML.
  • Vega-lite - Para quem precisa visualizar dados.
  • Emojis - Para colocar um pouco mais de expressão nos seus documentos.

O site oficial do CodiMD fica em https://github.com/hackmdio/codimd 

Suporte ao Markdown no CodiMD

A Tabela 1.1 mostra os elementos Markdown compatíveis com o CodiMD.

Elemento Suporte Obs
Títulos Sim A ferramenta não oferece suporte a sintaxe alternativa para construir títulos/cabeçalhos no Markdown.
Parágrafos Sim
Quebras de linha Sim Usando o espaço em branco à direita do conteúdo para quebrar linhas pode gerar problemas de formatação no CodiMD, mas o nosso querido amigo Enter ainda é compatível, basta pressionar a tecla para inserir uma nova linha.
Negrito Sim
Itálico Sim
Bloco de citação Sim Aninhar os blocos de citação também pode resultar em erros de formatação, o CodiMD não é compatível com blocos de citação aninhados.
