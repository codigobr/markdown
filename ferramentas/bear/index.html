Bear

O Bear é um aplicativo para macOS e iOS projetado para uma coisa: anotações. É como o Evernote, mas sem o inchaço.

  1. Markdown
  3. Ferramentas

Visão geral

Não há muitos recursos de whizbang no Bear. Em vez disso, o Bear cumpre consistentemente todas as suas promessas. Tags, pesquisa e sincronização funcionam perfeitamente. O aplicativo é bem intuitivo para o usuário, esse é um dos pontos fortes da ferramenta.

Site: https://bear.app/

Markdown no aplicativo Bear

O Bear não habilita automaticamente o suporte para Markdown, você deve fazer isso nas preferências. O aplicativo possui um editor em tempo real híbrido e um editor de texto - você pode ver a sintaxe do Markdown e a maneira como a formatação altera o texto. Demora um pouco para se acostumar, mas é útil se você está apenas começando com o Markdown.

Habilitando o suporte de Markdown

Para ativar o suporte ao Markdown no Bear, abra a janela de preferências. Na guia General, ative a configuração Markdown compatibility mode.

Dica: se você estiver usando o Bear em mais de um dispositivo, será necessário ativar a configuração do modo de compatibilidade com o Markdown em todos os seus dispositivos.

Compatibilidade do Bear com Markdown

O Bear fornece suporte para os seguintes elementos Markdown.

Elemento Suporte Obs
Títulos Parcial Não é compatível com a sintaxe alternativa
Parágrafos Sim
Quebras de linha Parcial Não há compatibilidade com a sintaxe Markdown de quebra de linha com um espaço em branco à direita, mas você pode pressionar o enter uma vez para obter o mesmo resultado.
Negrito Sim
Itálico Sim
Bloco de citação Parcial Blocos de citação aninhados não são suportados.
Listas ordenadas Sim
Listas não ordenadas Sim
Código Sim
Linhas horizontais Sim
Links Sim Você pode criar uma vinculação colocando o nome da nota entre dois colchetes (exemplo, [[nome-da-nota]]).
Imagens Não A sintaxe Markdown não é suportada, mas você ainda pode arrastar e soltar imagens em uma anotação.
Tabelas Não
Blocos de código cercado Sim O realce da sintaxe também é suportado.
Notas de rodapé Não
Id de títulos Parcial Gerado automaticamente. Não há como definir identificadores de títulos personalizados. Você pode copiar um link para um cabeçalho clicando no símbolo H# próximo ao título rente a margem e selecionar Copy Link to Here. Veja o a documentação para mais informações.
Listas de definições Não
Riscado Sim
Listas de tarefas Sim
Emoji (copiar e colar) Sim
Emoji (códigos de acesso) Sim
Link automático de URL Sim
Desativar o link automático de URL Sim
HTML Parcial O HTML não é renderizado nas notas do Bear, mas aparentemente é renderizado na hora da exportação.
Suporte para elementos de sintaxe adicionais

O Bear também é compatível com alguns elementos obscuros.

Elemento Markdown Saída renderizada
Realçar ::palavra ou frase:: palavra ou frase
Sublinhado ~palavra ou frase~ palavra ou frase
