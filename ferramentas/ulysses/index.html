Ulysses

O Ulysses é um dos aplicativos mais aclamados por escritores que precisam de recursos profissionais para criação de documentos em dispositivos Mac, iPad e iPhone.

  1. Markdown
  3. Editores Markdown
  5. Ulysses

Visão geral

O aplicativo número um de escritores e revisores que usam os dispositivos da Apple possui opções de temas e exportação de arquivos difícil de encontrar em outros programas.

Porém, quando falamos em Markdown, a ferramenta deixa muito a desejar.

O suporte ao Markdown no Ulysses fica restrito a um subconjunto da sintaxe da linguagem, mas mesmo assim, alguns elementos considerados como básicos não estão presentes.

Outro fator, no mínimo peculiar, é o uso de notações que não são padrão do Markdown para criar elementos do... Markdown?

Por esses motivos, utilizar o Ulysses para editar documentos Markdown pode se tornar uma tarefa frustrante, se você está apenas iniciando ou não quer aprender uma nova sintaxe da linguagem, é melhor usar outro programa.

O site oficial do Ulysses fica em https://ulysses.app/ .

Suporte ao Markdown no Ulysses

A Tabela 1.1 mostra os elementos Markdown compatíveis com o Ulysses, atente-se para as observações de alguns elementos para entender como construí-los usando a sintaxe alternativa da ferramenta.

Elemento Suporte Obs
Títulos Sim O Ulysses não fornece compatibilidade com a sintaxe alternativa para a criação de títulos.
Parágrafos Sim
Quebras de linha Não
Negrito Sim
Itálico Sim
Bloco de citação Sim
Listas ordenadas Sim
Listas não ordenadas Sim
Código Parcial Os blocos de código NÃO são suportados pela ferramenta.
Linhas horizontais Parcial Para formar uma linha horizontal no Ulysses é necessário usar quatro traços ----.
Links Não O Ulysses NÃO é compatível com a sintaxe do Markdown para inserir links, mas você pode usar o menu Markup para executar a mesma tarefa.
Imagens Não A sintaxe do Markdown também NÃO é compatível com o programa, use o menu Markup para inserir imagens no seu documento.
Tabelas Não
Blocos de código cercado Não Use o menu Markup para inserir blocos de código no Ulysses, a sintaxe do Markdown NÃO é suportada.
Notas de rodapé Não Semelhante a vários elementos, esse também deve ser inserido usando o menu Markup, a sintaxe do Markdown para notas de rodapé, infelizmente, não é suportada pelo app.
Id de títulos Não
Listas de definições Não
Riscado Parcial Para riscar um texto no Ulysses utilize duas barras verticais ao redor do conteúdo, por exemplo: ||texto riscado||. A sintaxe padrão do Markdown NÃO é suportada.
Listas de tarefas Sim A criação de listas de tarefas no Ulysses pode ser feita usando somente os colchetes [], sem a necessidade de adicionar um traço - no início da linha.
Emoji (copiar e colar) Desconhecido
Emoji (códigos de acesso) Desconhecido
Link automático de URL Não
Desativar o link automático de URL Sim Não é preciso pois o Ulysses não vincula automaticamente os links.
HTML Não
