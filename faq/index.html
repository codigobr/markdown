    O que é o Markdown?

    Markdown é uma linguagem de marcação usada para formatar diversos tipos de documento, extremamente fácil de usar, sua formatação é tão simples que um texto estilizado com Markdown nem parece que possui marcação nele.
  • question_answer

    Como Converter Um Arquivo Markdown Em HTML?

    Para converter um documento formatado em Markdown para HTML use uma ferramenta como o Pandoc ou um editor online como o Markedit
  • question_answer

    Como formatar um parágrafo em Markdown?

    Para formatar um parágrafo no Markdown, separe o conteúdo do parágrafo por uma linha em branco antes e depois.

    Exemplo:

    Esse conteúdo pertence ao primeiro parágrafo.

E esse daqui, pertence ao segundo.
