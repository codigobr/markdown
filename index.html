Guia completo do Markdown Para Iniciantes

Guia de referência gratuito que explica como usar o Markdown.

  1. Markdown

O que é o Markdown?

O Markdown é uma linguagem de marcação simples e fácil de usar que você pode formatar praticamente qualquer documento.

Aprenda os conceitos.

Comece a usar o Markdown agora mesmo seguindo o guia de Introdução. Ele foi projetado para todos, até para iniciantes.

Mergulhe na sintaxe.

Seja você um iniciante no Markdown ou um profissional experiente, suas dúvidas sobre formatação e elementos serão sanadas na página de sintaxe básica.

Avançando um pouquinho.

Precisa de mais elementos em Markdown? Sem problemas, usando a sintaxe estendida você pode criar tabelas, blocos de códigos, links automáticos e muito mais.
