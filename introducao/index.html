O que é o Markdown?

Markdown é uma linguagem de marcação leve que você pode usar para adicionar elementos de formatação a documentos de texto sem formatação. Criado por John Gruber em 2004, o Markdown agora é uma das linguagens de marcação mais populares do mundo.

O uso do Markdown é diferente do uso de um editor WYSIWYG. Em um aplicativo como o Microsoft Word, você clica nos botões para formatar palavras e frases, e as alterações são visíveis imediatamente. Markdown não é assim. Ao criar um arquivo no Markdown, você adiciona a sintaxe da linguagem ao texto para indicar quais palavras e frases devem ser formatadas.

Por exemplo, para indicar um cabeçalho, você adiciona um sinal de número antes dele (# Título Um). Ou, para deixar uma frase em negrito, adicione dois asteriscos antes e depois dela (**este texto está em negrito**). Pode demorar um pouco para se acostumar com a sintaxe Markdown, especialmente se você estiver acostumado com aplicativos WYSIWYG. A captura de tela abaixo mostra um arquivo Markdown exibido no editor de texto Atom.

Você pode adicionar elementos de formatação do Markdown a um arquivo de texto sem formatação usando um aplicativo de editor de texto. Ou você pode usar um dos muitos aplicativos Markdown para sistemas operacionais macOS, Windows, Linux, iOS e Android. Existem também vários aplicativos baseados na Web projetados especificamente para gravação no Markdown.

Dependendo do aplicativo usado, talvez não seja possível visualizar o documento formatado em tempo real. Mas está tudo bem. De acordo com Gruber, a sintaxe do Markdown foi projetada para ser legível e discreta, para que o texto nos arquivos do Markdown possa ser lido mesmo que não seja renderizado.

O objetivo principal do design da sintaxe de formatação do Markdown é torná-lo o mais legível possível. A idéia é que um documento no formato Markdown seja publicável como está, como texto sem formatação, sem parecer que foi marcado com tags ou instruções de formatação.

Por que usar o Markdown?

Você pode estar se perguntando por que as pessoas usam o Markdown em vez de um editor WYSIWYG. Por que escrever com o Markdown quando você pode pressionar botões em uma interface para formatar seu texto? Como se vê, existem algumas razões diferentes pelas quais as pessoas usam o Markdown em vez dos editores do WYSIWYG.

  • Markdown pode ser usado para tudo. As pessoas o usam para criar sites, documentos, notas, livros, apresentações, mensagens de e-mail e documentação técnica.
  • Markdown é portátil. Arquivos que contêm texto no formato Markdown podem ser abertos usando praticamente qualquer aplicativo. Se você decidir que não gosta do aplicativo Markdown em uso no momento, poderá importar seus arquivos Markdown para outro aplicativo Markdown. Isso contrasta fortemente com aplicativos de processamento de texto como o Microsoft Word que bloqueiam seu conteúdo em um formato de arquivo proprietário.
  • O Markdown é independente da plataforma. Você pode criar texto no formato Markdown em qualquer dispositivo executando qualquer sistema operacional.
  • Markdown é uma prova do futuro. Mesmo que o aplicativo que você está usando pare de funcionar em algum momento no futuro, você ainda poderá ler o texto no formato Markdown usando um aplicativo de edição de texto. Essa é uma consideração importante quando se trata de livros, teses universitárias e outros documentos importantes que precisam ser preservados indefinidamente.
  • Markdown está em todo lugar. Sites como o Reddit e o GitHub suportam o Markdown, e muitos aplicativos de desktop e baseados na Web o suportam.

Pegando no tranco

A melhor maneira de começar a usar o Markdown é usá-lo. Isso é mais fácil do que nunca, graças a uma variedade de ferramentas gratuitas.

Você nem precisa baixar nada. Existem vários editores online do Markdown que você pode usar para tentar escrever no Markdown. Dillinger é um dos melhores editores online do Markdown. Basta abrir o site e começar a digitar no painel esquerdo. Uma visualização do documento renderizado aparece no painel direito.

Você provavelmente manterá o site da Dillinger aberto enquanto lê este guia. Dessa forma, você pode tentar a sintaxe ao aprender sobre ela. Depois de se familiarizar com o Markdown, convém usar um aplicativo Markdown que possa ser instalado no seu computador desktop ou dispositivo móvel.

Como funciona?

Dillinger facilita a escrita no Markdown porque esconde o que está acontecendo nos bastidores, mas vale a pena explorar como o processo funciona em geral.

Quando você escreve no Markdown, o texto é armazenado em um arquivo de texto sem formatação que possui uma extensão .md ou .markdown. Mas então o que? Como seu arquivo no formato Markdown é convertido em HTML ou em um documento pronto para impressão?

A resposta curta é que você precisa de um aplicativo Markdown capaz de processar o arquivo Markdown. Existem muitos aplicativos disponíveis - tudo, desde scripts simples até aplicativos de desktop que se parecem com o Microsoft Word. Apesar de suas diferenças visuais, todos os aplicativos fazem a mesma coisa. Como Dillinger, todos convertem o texto no formato Markdown em HTML para que ele possa ser exibido nos navegadores da web.

Os aplicativos Markdown usam algo chamado processador Markdown (também conhecido como "analisador" ou "implementação") para pegar o texto no formato Markdown e enviá-lo para o formato HTML. Nesse ponto, seu documento pode ser visualizado em um navegador da Web ou combinado com uma folha de estilos e impresso. Você pode ver uma representação visual desse processo abaixo.

Nota: O aplicativo e o processador Markdown são dois componentes separados. Por uma questão de brevidade, eu os combinei em um elemento ("Markdown App") na figura abaixo.

Para resumir, este é um processo de quatro partes:

  • Crie um arquivo Markdown usando um editor de texto ou um aplicativo Markdown dedicado. O arquivo deve ter uma extensão .md ou .markdown.
  • Abra o arquivo Markdown em um aplicativo Markdown.
  • Use o aplicativo Markdown para converter o arquivo Markdown em um documento HTML.
  • Visualize o arquivo HTML em um navegador da Web ou use o aplicativo Markdown para convertê-lo em outro formato de arquivo, como PDF.
Da sua perspectiva, o processo variará um pouco, dependendo do aplicativo que você usa. Por exemplo, o Dillinger combina essencialmente as etapas de 1 a 3 em uma interface única e integrada - tudo o que você precisa fazer é digitar no painel esquerdo e a saída renderizada magicamente aparece no painel direito. Mas se você usar outras ferramentas, como um editor de texto com um gerador de site estático, verá que o processo é muito mais visível.

Para que serve o Markdown?

O Markdown é uma maneira rápida e fácil de fazer anotações, criar conteúdo para um site e produzir documentos prontos para impressão.

Não leva muito tempo para aprender a sintaxe do Markdown, e depois que você souber como usá-lo, poderá escrever usando o Markdown em qualquer lugar. A maioria das pessoas usa o Markdown para criar conteúdo para a web, mas o Markdown é bom para formatar tudo, desde mensagens de email a lista de compras.

Aqui estão alguns exemplos do que você pode fazer com o Markdown.

Websites

O Markdown foi desenvolvido para a Web, portanto, não é de surpreender que existam muitos aplicativos projetados especificamente para a criação de conteúdo de sites.

Se você está procurando a maneira mais simples possível de criar um site com arquivos Markdown, consulte blot.im e smallvictori.es. Depois de se inscrever em um desses serviços, eles criam uma pasta Dropbox no seu computador. Basta arrastar e soltar seus arquivos do Markdown na pasta e - puf! - eles estão no seu site. Não poderia ser mais fácil.

Se você está familiarizado com HTML, CSS e controle de versão, confira o Jekyll, um popular gerador de site estático que pega arquivos do Markdown e cria um site em HTML. Uma vantagem dessa abordagem é que o GitHub Pages fornece hospedagem gratuita para sites gerados pelo Jekyll. Se Jekyll não é sua xícara de chá, basta escolher um dos muitos outros geradores de sites estáticos disponíveis.

Se você deseja usar um sistema de gerenciamento de conteúdo (CMS) para alimentar seu site, dê uma olhada no Ghost. É uma plataforma de blog gratuito e de código aberto com um bom editor de Markdown. Se você é um usuário do WordPress, ficará feliz em saber que há suporte ao Markdown para sites hospedados no WordPress.com. Sites WordPress auto-hospedados podem usar o plugin Jetpack.

Documentos

O Markdown não possui todos os sinos e assobios de processadores de texto como o Microsoft Word, mas é bom o suficiente para criar documentos básicos, como tarefas e cartas. Você pode usar um aplicativo de criação de documentos do Markdown para criar e exportar documentos no formato Markdown para o formato de arquivo PDF ou HTML. A parte do PDF é essencial, pois depois de ter um documento PDF, você pode fazer qualquer coisa com ele - imprimi-lo, enviá-lo por e-mail ou enviá-lo para um site.

Aqui estão alguns aplicativos de criação de documentos do Markdown que eu recomendo:

  • Mac: MacDown, iA Writer ou Marked
  • iOS / Android: iA Writer
  • Windows: ghostwriter ou Markdown Monster
  • Linux: ReText ou ghostwriter
  • Web: Dillinger ou StackEdit

Dica: O iA Writer fornece modelos para visualizar, imprimir e exportar documentos no formato Markdown. Por exemplo, o modelo "Academic - MLA Style" recua parágrafos e adiciona espaçamento entre frases duplas.

Notas

Em quase todos os aspectos, o Markdown é a sintaxe ideal para fazer anotações. Infelizmente, o Evernote e o OneNote, dois dos aplicativos de anotação mais populares, atualmente não oferecem suporte ao Markdown. A boa notícia é que várias outras aplicações de notas fazer suporte Markdown:

  • Simplenote é um aplicativo gratuito de anotações de barebones disponível para todas as plataformas.
  • Notable é um aplicativo de anotações executado em uma variedade de plataformas.
  • O Bear é um aplicativo semelhante ao Evernote disponível para dispositivos Mac e iOS. Ele não usa o Markdown exclusivamente por padrão, mas você pode ativar o modo de compatibilidade do Markdown.
  • O Boostnote se autodenomina como um "aplicativo de anotações de código aberto desenvolvido para programadores".

Se você não pode participar do Evernote, consulte o Marxico, um editor de Markdown baseado em assinatura do Evernote ou use o Markdown Here no site do Evernote.

Livros

Procurando publicar automaticamente um romance? Experimente o Leanpub , um serviço que pega seus arquivos no formato Markdown e os transforma em um livro eletrônico. O Leanpub produz seu livro no formato de arquivo PDF, EPUB e MOBI. Se você deseja criar cópias em brochura do seu livro, pode carregar o arquivo PDF em outro serviço, como o Kindle Direct Publishing. Para saber mais sobre como escrever e publicar um livro usando o Markdown, leia esta postagem do blog.

Apresentações

Acredite ou não, você pode gerar apresentações a partir de arquivos no formato Markdown. A criação de apresentações no Markdown demora um pouco para se acostumar, mas depois que você pega o jeito, é muito mais rápido e fácil do que usar um aplicativo como o PowerPoint ou o Keynote. O Remark (projeto GitHub) é uma ferramenta popular de apresentação de slides do Markdown, baseada no navegador, assim como o Cleaver (projeto GitHub). Se você usa um Mac e prefere usar um aplicativo, consulte Deckset ou Marked.

E-mail

Se você envia muitos e-mails e está cansado dos controles de formatação disponíveis na maioria dos sites de provedores de e-mail, ficará feliz em saber que existe uma maneira fácil de escrever mensagens de e-mail usando o Markdown. Markdown Here é uma extensão de navegador gratuita e de código aberto que converte texto formatado em Markdown em HTML pronto para envio.

Documentação

Markdown é um ajuste natural para documentação técnica. Empresas como o GitHub estão cada vez mais migrando para o Markdown para obter a documentação - confira em seu blog como migraram sua documentação no formato Markdown para o Jekyll. Se você escrever documentação para um produto ou serviço, dê uma olhada nessas ferramentas úteis:

  • Read The Docs pode gerar um site de documentação a partir dos seus arquivos Markdown de código aberto. Basta conectar seu repositório GitHub ao serviço e pressionar - Read the Docs faz o resto. Eles também têm um serviço para entidades comerciais.
  • O MkDocs é um gerador de site estático rápido e simples, voltado para a documentação do projeto de construção. Os arquivos de origem da documentação são gravados no Markdown e configurados com um único arquivo de configuração YAML. O MkDocs possui vários temas internos, incluindo uma adaptação tema Read The Docs para uso com o MkDocs. Um dos temas mais recentes é o MkDocs Material.
  • O Docusaurus é um gerador de site estático projetado exclusivamente para criar sites de documentação. Ele suporta traduções, pesquisa e controle de versão.
  • O VuePress é um gerador de site estático desenvolvido com o Vue e otimizado para escrever documentação técnica.
  • Jekyll foi mencionado anteriormente na seção em sites, mas também é uma boa opção para gerar um site de documentação a partir de arquivos Markdown. Se você seguir esse caminho, verifique o tema da documentação do Jekyll.

Sabores do Markdown

Um dos aspectos mais confusos do uso do Markdown é que praticamente todos os aplicativos do Markdown implementam uma versão ligeiramente diferente do Markdown. Essas variantes do Markdown são comumente chamadas de sabores. É seu trabalho dominar qualquer sabor do Markdown que seu aplicativo tenha implementado.

Para entender o conceito de sabores Markdown, pode ser útil pensar neles como dialetos linguísticos. As pessoas em Ciudad Juárez falam espanhol, assim como as pessoas em Barcelona, mas há diferenças substanciais entre os dialetos usados nas duas cidades. O mesmo vale para pessoas que usam diferentes aplicativos Markdown. Usar Dillinger para escrever com Markdown é uma experiência muito diferente do que usar Ulisses.

Na prática, isso significa que você nunca sabe exatamente o que significa quando uma empresa diz que é compatível com o "Markdown". Eles estão falando apenas dos elementos básicos da sintaxe, ou de todos os elementos básicos e estendidos da sintaxe combinados, ou de alguma combinação arbitrária dos elementos da sintaxe? Você não saberá até ler a documentação ou começar a usar o aplicativo.

Se você está apenas começando, o melhor conselho que posso dar é escolher um aplicativo com um bom suporte ao Markdown, como esse aqui. Isso ajudará muito a manter a portabilidade dos seus arquivos Markdown. Convém armazenar e usar os arquivos do Markdown em outros aplicativos e, para isso, é necessário começar com um aplicativo que ofereça um bom suporte. Você pode usar o nosso diretório de ferramentas para encontrar um aplicativo que atenda à conta.
