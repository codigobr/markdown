Visão geral

Precisa refrescar a memória em relação a sintaxe do Markdown? Ótimo!

Abaixo são listados os elementos mais comuns e utilizados da linguagem.

Vale ressaltar que, aqui somente o básico do Markdown é apresentado, se precisar de mais informações consulte as seções de sintaxe básica e sintaxe estendida.

Sintaxe básica

Os elementos que fundamentam o Markdown são mostrados abaixo. Esse elementos de formatação foram descritos no documento original da linguagem e a grande maioria dos programas fornecem suporte eles.

Elemento Markdown
Títulos 
# H1
## H2
### H3
Negrito 
**texto em negrito**
Itálico 
*texto em itálico*
Blockquote 
> Blockquote
Lista ordenada 
1. Primeiro elemento
2. Segundo elemento
3. Terceiro elemento
Lista não ordenada 
- Primeiro item
- Segundo item
- Terceiro item
Código 
`código`
Linha horizontal 
---
Link 
[título](https://markdown.net.br)
Imagem 
![texto alternativo](https://markdown.net.br/assets/img/markdown-logo-transparent.svg)

Sintaxe Estendida

Com a evolução do Markdown outros elementos foram implementados enriquecendo o leque de opções para formatar documentos. Por um lado, um pouco mais de complexidade, mas por outro, a linguagem ganhou mais recursos para a construções de documentos diversos.

Tenha em mente que, nem todos os aplicativos desenvolvidos para Markdown dão suporte à esses elementos, é sempre bom consultar a documentação do programa ou a seção de ferramentas.
